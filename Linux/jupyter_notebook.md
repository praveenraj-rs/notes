### Keyboard Shortcuts

| Action                             | Shortcut                   |
|------------------------------------|----------------------------|
| Run current cell                   | Ctrl + Enter               |
| Run cell and select below          | Shift + Enter               |
| Run cell and insert below          | Alt + Enter                 |
| Insert cell above                  | A                          |
| Insert cell below                  | B                          |
| Delete selected cells              | D, D (press D twice)       |
| Save notebook                      | Ctrl + S                   |
| Change cell type to code           | Y                          |
| Change cell type to markdown       | M                          |
| Toggle line numbers                | L                          |
| Interrupt kernel                   | I, I (press I twice)       |
| Restart kernel                     | 0, 0 (press 0 twice)       |
| Command palette                    | P                          |
| Help                               | H                          |
