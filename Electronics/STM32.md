### **General-Purpose Input/Output (GPIO) Overview**

#### **1. Microcontroller Pins & Ports**

- Microcontroller pins are grouped into **ports** (e.g., Port A, Port B, etc.).
    
- Pins are referenced as **Px.y** (e.g., **P1.1** = Port A, Pin 1; **PD7** = Port D, Pin 7).
    

#### **2. General-Purpose I/O (GPIO)**

- **GPIO (General-Purpose Input/Output)** pins can be configured as:
    
    - **General-purpose pins** (default).
        
    - **Alternate function pins** (used for peripherals like UART, ADC, etc.).
        
- Example:
    
    - UART uses GPIO pins for **TX (transmit) and RX (receive)**.
        
    - ADC requires a pin to be set as **analog input** instead of a standard GPIO.
        

#### **3. GPIO Registers**

Every microcontroller's GPIO module has at least **two registers**:

- **Direction Register** → Configures a pin as **input** or **output**.
    
- **Data Register** → Used to **read/write** data to/from the pin.
    
- Some microcontrollers (e.g., STM) may name these registers differently.
    

#### **4. Microcontroller Buses**

- Two types of buses inside a microcontroller:
    
    1. **AHB (Advanced High-Performance Bus)** → Faster, single clock cycle access.
        
    2. **APB (Advanced Peripheral Bus)** → Slower, requires at least two clock cycles.
        
- Some peripherals can be accessed via **both** buses (e.g., GPIO in Texas Instruments MCUs).
    

#### **5. Microcontroller Clock Sources**

Three main clock sources:

1. **On-chip RC Oscillator** (Resistor-Capacitor) → Least precise.
    
2. **External Crystal Oscillator** → More precise than RC oscillator.
    
3. **PLL (Phase-Locked Loop)** → Can **increase or decrease** the clock frequency.
    

#### **6. Summary**

- **Microcontroller pins are arranged in ports** (e.g., P1.1 → Port A, Pin 1).
    
- **GPIO pins can be configured for general-purpose or alternate functions** (e.g., UART, ADC).
    
- **GPIO has two key registers**: Direction Register (input/output) & Data Register (read/write).
    
- **Microcontrollers have two buses**: AHB (faster) & APB (slower).
    
- **Clock sources** include **RC Oscillator, External Crystal, and PLL** for different precision levels.
    

### **Next Steps**

- Explore **microcontroller documentation** for GPIO configurations.
